# 2018-05-31之前的Android SDK更换说明

说明

2018-5-31之前提供的Android SDK是提供的安卓工程接入方式。
鉴于对接便捷性和维护性的考虑，2018-06-01日起调整为远程依赖接入的方式。

调整将花费您3-5分钟时间，感谢您的配合和支持~

主要调整内容：sdk4.21lib本地依赖替换为远程依赖加载。（仅为依赖及资源配置调整，其他均无变动）

# 具体调整步骤

  1. 拿到SDK平台提供最新的SDK压缩包（包括参数、资源等）

  2. 将游戏工程里面的build.gradle文件中的依赖 sdk4.2lib 去掉，替换为sdk远程依赖 compile 'com.myths.sdkbase:sdkBase:1.0.3'，
    在此文件末尾加入apply plugin: 'com.google.gms.google-services' 。

dependencies {
    // compile project(':sdk4.2lib') 删除改行
    compile 'com.myths.sdkbase:sdkBase:1.0.3' //  新加入
}
apply plugin: 'com.google.gms.google-services' // 新加入

  1. 在根目录的build.gradle文件中加入 classpath 'com.google.gms:google-services:3.2.0'

  2. 将sdk提供的google-services.json放入游戏工程根目录下

  3. 将sdk提供的drawable-xxhdpi目录下面的资源文件复制到游戏主工程的drawable-xxhdpi目录下

其他内容均无变动

详见 Android接入文档
